⚙ Kết nối Google Sheets
Dán 1 Apps Script URL duy nhất — Google Sheet tổng hợp gồm 3 tab: Inovance, Yaskawa, Mitsubishi.
▶ Hướng dẫn tạo Google Sheet & Apps Script (click để mở)
Bước 1 — Tạo Google Sheet tổng hợp:
Tạo 1 Google Sheet mới → đổi tên 3 tab lần lượt thành: Inovance  |  Yaskawa  |  Mitsubishi
Copy toàn bộ dữ liệu bảng giá vào từng tab (giữ nguyên cấu trúc cột như bảng gốc).

Bước 2 — Tạo Apps Script:
Mở Google Sheet → Extensions → Apps Script → xóa code cũ → dán code sau: 
function doGet(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var names = ['Inovance', 'Yaskawa', 'Mitsubishi'];
  var result = {};
  names.forEach(function(name) {
    var sheet = ss.getSheetByName(name);
    if (sheet) result[name] = sheet.getDataRange().getValues();
  });
  var json = JSON.stringify(result);
  var cb = e && e.parameter && e.parameter.callback;
  if (cb) {
    return ContentService
      .createTextOutput(cb + '(' + json + ')')
      .setMimeType(ContentService.MimeType.JAVASCRIPT);
  }
  return ContentService
    .createTextOutput(json)
    .setMimeType(ContentService.MimeType.JSON);
}
Bước 3 — Deploy:
Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy
Copy URL dạng https://script.google.com/macros/s/…/exec → dán vào ô bên dưới.
💡 Chỉ cần 1 URL duy nhất cho cả 3 bảng giá. App tự lưu URL — mở lại không cần nhập lại.
Chưa kết nối — nhập URL và nhấn Kết nối
Inovance — Tìm theo Model
Tìm gần đúng — nhập 1 phần model. VD: MD500, H5U, IS620
Yaskawa — Tìm theo Short Name
Tìm gần đúng — nhập 1 phần tên. VD: GA700, SGD7S, CACR
Mitsubishi — Tìm theo Material Name
Tìm gần đúng — nhập 1 phần model. VD: AC30TB, A6BAT, AJ65
