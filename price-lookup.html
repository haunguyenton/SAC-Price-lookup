⚙ Kết nối dữ liệu Google Sheets
Dán Apps Script Web App URL của từng sheet vào bên dưới.
Xem hướng dẫn tạo URL: ▶ Hướng dẫn tạo Apps Script (click để mở)
Làm 1 lần cho mỗi sheet:
1. Mở Google Sheet → menu Extensions → Apps Script
2. Xóa code cũ, dán đoạn code sau:
function doGet(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheets = ss.getSheets();
  var result = [];
  sheets.forEach(function(sheet) {
    var data = sheet.getDataRange().getValues();
    result.push({ sheetName: sheet.getName(), data: data });
  });
  var json = JSON.stringify(result);
  var cb = e && e.parameter && e.parameter.callback;
  if(cb) {
    return ContentService
      .createTextOutput(cb + '(' + json + ')')
      .setMimeType(ContentService.MimeType.JAVASCRIPT);
  }
  return ContentService
    .createTextOutput(json)
    .setMimeType(ContentService.MimeType.JSON);
}
3. Nhấn Deploy → New deployment
4. Type: Web app | Execute as: Me | Who has access: Anyone
5. Nhấn Deploy → copy URL dạng https://script.google.com/macros/s/…/exec
6. Dán vào ô bên dưới
💡 URL Apps Script chỉ cần tạo 1 lần. Lần sau mở app chỉ cần dán lại URL (hoặc lưu vào bookmark). Dữ liệu được tải trực tiếp từ sheet — luôn cập nhật.
